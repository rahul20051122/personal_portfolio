## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/rahul20051122/personal_portfolio.git
```

Navigate to project directory:

```bash
cd personal_portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
